PVLib Physics Chain — Production Implementation
PYTHON · PVLIB
# Step 1: Solar position (ephemeris)
solar_pos = pvlib.solarposition.get_solarposition(times, lat, lon, altitude)

# Step 2: Clear-sky irradiance (Ineichen model + Linke turbidity)
clearsky = loc.get_clearsky(times, model='ineichen', linke_turbidity=linke_TL)

# Step 3: Cloud correction via kt (clearness index from satellite)
GHI_eff = clearsky['ghi'] × kt_satellite # kt from INSAT-3DR Cloud OD → Heliosat-2

# Step 4: Decompose GHI → DNI, DHI (Erbs/Dirint)
dni, dhi = pvlib.irradiance.dirint(GHI_eff, solar_pos['zenith'], times)

# Step 5: POA irradiance (Perez transposition for tilted plane)
poa = pvlib.irradiance.get_total_irradiance(tilt, azimuth, solar_pos, dni, dhi, GHI_eff)

# Step 6: Cell temperature (Faiman model)
T_cell = pvlib.temperature.faiman(poa['poa_global'], T_amb, wind_speed)

# Step 7: DC power (Sandia PVWatts)
P_dc = η_panel × A_panel × poa['poa_global'] × (1 + γ × (T_cell − 25))

# Step 8: AC power (inverter model: NREL CEC efficiency curve)
P_ac = pvlib.inverter.sandia(P_dc, V_dc, inverter_params) × f_soiling × f_age

Cooling Load Model — Engineering Detail
PHYSICS-ML HYBRID
Model Equation
L_cool(t) = N_AC(t) · P_rated · COP_adj
· OccRate(h,dow) · f_thermal

N_AC(t) = N_0 · exp(r·t) + ε_survey(t)
# r ≈ 0.12/yr (Rajasthan 2019–2024)

COP_adj = COP_rated × (1 − 0.03·(T−25))
× (1 − 0.005·(RH−50))

f_thermal = 1 − exp(−t/τ_thermal)
# τ_thermal ≈ 45 min (thermal mass delay)
Calibration Sources
NCRB consumer survey data (annual) → N_AC by substation zone
BEE star-rating distribution → average P_rated and COP_rated per zone
Smart meter 15-min data → OccRate(h,dow) occupancy profile per consumer segment
Building stock census (SECC) → thermal mass correction per ward
Quarterly recalibration via billing regression; Δ flagged if N_AC drift >5%

MATH
σ²(D_net) = σ²(D_gross) + σ²(P_DRE) − 2·ρ·σ(D_gross)·σ(P_DRE)

# ρ = corr(D_gross, P_DRE) via weather; typically 0.3–0.5 (same temperature driver)
